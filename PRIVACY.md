# MarkClip Privacy Policy

Last updated: 2026-05-26

MarkClip converts web page content to Markdown locally in your browser.

## Data Collection

MarkClip does not collect, store, sell, transmit, or share personal data.

MarkClip does not upload page content, browsing history, generated Markdown, or clipboard contents to any server.

## Local Processing

All page extraction and Markdown conversion happen locally on the user's device.

MarkClip reads page content only when the user actively uses the extension, such as clicking copy, download, refresh, or using the floating panel.

## Permissions

MarkClip uses Chrome extension permissions only for its core Markdown export features:

- `activeTab`: Access the current tab when the user invokes the extension.
- `scripting`: Inject local extension scripts and bundled conversion libraries.
- `clipboardWrite`: Copy generated Markdown when the user clicks copy.
- `storage`: Save local preferences, including theme, export mode, image-link removal, floating button visibility, and floating button position.
- `host_permissions`: Display the floating panel on ordinary web pages and read page content for local conversion.

## Third-Party Services

MarkClip does not use a remote backend or third-party analytics service.

Bundled third-party libraries, including Turndown and Mozilla Readability, run locally inside the browser extension.

## Contact

For privacy questions, contact the project maintainer through the GitHub repository.
